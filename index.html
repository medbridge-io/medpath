THE JOURNEY · 4단계 경로

기초에서 실전까지 · 4단계 경로

MedBridge로 영어 기초를 세운 뒤, MedPath의 3 단계가 같은 분자와 같은 말하기 기술을 “무엇인가 → 어떻게 작동하나 → 왜 필수적인가”로 끌어올립니다. 단계마다 말하기가 한 차원씩 자랍니다.
같은 토대 위에서, 한 단계씩 더 깊이.
0 · FOUNDATION

MedBridge

의학 영어, 입을 여는 준비

의학 영어의 핵심 어휘·발음·기본 문장 구조를 익히고, 짧은 문장으로 소리 내어 말하는 습관을 만듭니다. 3개월 · 주 2·3·4회 · 30개 토픽.

1 · MEDPATH

Starter

정의 — 무엇인가? (What)

대상을 정확히 명명하고 한 문장으로 정의하는 힘. 50레슨의 토대를 정의 수준에서 완성합니다.

2 · MEDPATH

Core

설명 — 어떻게 작동하나? (How)

원인과 결과의 메커니즘을 영어로 설명하는 힘. 같은 50레슨을 설명 수준으로 다시 다룹니다.

3 · MEDPATH

Pro

논증 — 왜 필수적인가? (Why)

통합·논증과 즉흥 대응. 면접관 앞에서 압박 속에 답하는 구술시험 실전력을 완성합니다.

전체 경로 = 4단계 × 3개월 · 주 4회 — 총 약 12개월 (MedPath 3티어는 9개월)

MedPath의 3 티어는 아래 50개 레슨·구조·레이아웃을 동일하게 공유합니다. 레슨의 주제·어휘·순서는 그대로 유지되며, 학생은 같은 내용을 한 단계씩 더 깊은 사고로 다시 다룹니다: 정의(What) → 설명(How) → 논증(Why). 각 레슨은 고정된 50분 흐름과 3종 교재(Class · Student · Teacher’s Book)로 나선형으로 연결됩니다.
